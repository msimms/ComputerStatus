Home Computer Status is a combination of website and application (python script) for remotely monitoring the health of a computer. Currently, it can monitor CPU usage, memory usage, as well as GPU utilization and temperature (currently Nvidia GPUs only).

To run it, get a copy of the python client script from github.com and run it on the computer you wish to monitor (you'll also need to have python installed, of course).

Since the client software is distributed as a python script, it will run on most commonly available operating systems. This includes Windows, Linux, and Mac OS.

The software is released under the MIT license, which is a very permissive license for open source software.
